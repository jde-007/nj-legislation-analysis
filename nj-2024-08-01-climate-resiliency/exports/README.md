# NJ Climate Resiliency Hearing Exports

## Files
- `speakers.csv/tsv` - All witnesses with credentials and speaker type
- `scientific_claims.csv/tsv` - 35 claims with exact quotes, page refs, evidence grades
- `legislative_items.csv/tsv` - 13 legislative items discussed
- `committee_members.csv` - Senate and Assembly committee members present

## Evidence Grades
- **strong** - Peer-reviewed, government data, replicated findings
- **moderate** - Single studies, industry data, recent events

## Categories
- temperature, sea_level, precipitation, storms, projections, economic

## Import Notes
- TSV files work best with Excel/Google Sheets (tab-separated)
- CSV files have standard comma separation
- Quotes contain commas - use proper CSV parsing
