This folder contains CSV files that can be imported into Excel or Google Sheets.

To create a multi-sheet workbook:
1. Open Google Sheets or Excel
2. Import each CSV as a separate sheet
3. Name sheets according to filename

Files:
- hearings.csv - Hearing metadata
- speakers.csv - All witness profiles
- scientific_claims.csv - 42 claims with page references
- legislative_outcomes.csv - 13 legislative items
- committee_members.csv - 8 legislators present
- quantitative_analysis.json - Full metrics (import to Sheets via Extensions > Import JSON)
